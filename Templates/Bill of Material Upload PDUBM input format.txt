Note Text (from ICFPMN)

Processed Flag must be equal to N. It will be set to Y when the record is successfully processed.

Numeric fields cannot contain commas, dashes instead of zeroes or blanks, or parentheses to signify a negative amount. An error message will be displayed in the log file (nnn is not a valid floating point value) and the data in the line with the error will not be uploaded.

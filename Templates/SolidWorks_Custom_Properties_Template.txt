SolidWorks Custom Properties Template for Expendable ERP Integration
================================================================

Copy and paste these custom properties into your SolidWorks parts for best results:

REQUIRED PROPERTIES:
-------------------
PartNo           = [Enter your part number/SKU]
Description      = [Enter part description]

RECOMMENDED PROPERTIES:
----------------------
Material         = [e.g., Steel, Aluminum, Plastic, etc.]
Weight           = [Part weight with units, e.g., "0.5 lbs"]
Cost             = [Unit cost, e.g., "12.50"]
Supplier         = [Preferred supplier name]
Category         = [Part category, e.g., "Fasteners", "Electronics"]
Finish           = [Surface finish if applicable]
DrawingNo        = [Drawing number if different from part number]

OPTIONAL PROPERTIES:
-------------------
Lead_Time        = [Lead time in days]
Min_Order_Qty    = [Minimum order quantity]
Package_Qty      = [Quantity per package]
Hazmat          = [Yes/No for hazardous materials]
Notes           = [Additional notes]

HOW TO ADD CUSTOM PROPERTIES IN SOLIDWORKS:
===========================================

1. Open your part or assembly in SolidWorks
2. Go to File > Properties
3. Click on the "Custom" tab
4. Click "Add" to create new properties
5. Enter the property name (e.g., "PartNo")
6. Select "Text" as the type
7. Enter the value
8. Click OK

TIPS:
=====
- Use consistent naming across all parts
- Keep descriptions clear and concise
- Include units for weight and dimensions
- Use standard material names
- Update properties when parts change

PROPERTY VALIDATION:
===================
The macro will use these fallback values if properties are missing:
- PartNo: Uses filename if not found
- Description: Uses filename if not found
- Other properties: Left blank if not found

BATCH PROPERTY UPDATES:
======================
For existing files, you can use SolidWorks Design Checker or 
third-party tools to batch update custom properties across 
multiple files.

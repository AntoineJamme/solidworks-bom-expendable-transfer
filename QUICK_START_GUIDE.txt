🚀 QUICK START GUIDE - SolidWorks to Expendable ERP
=====================================================

IMMEDIATE SETUP (5 minutes):
============================

1. INSTALL SOLIDWORKS MACRO:
   ✓ Copy "Scripts\SolidWorks_BOM_Export.swp" to your SolidWorks macros folder
   ✓ Default location: C:\Users\[YourName]\AppData\Roaming\SolidWorks\Macros\

2. CONFIGURE EXPENDABLE SETTINGS:
   ✓ Edit "config.json" - change these lines:
   
   "default_category": "Parts",         ← Change to your category
   "default_location": "Main Warehouse" ← Change to your warehouse

3. TEST THE SETUP:
   ✓ Open any assembly in SolidWorks
   ✓ Run the macro: Tools > Macro > Run > SolidWorks_BOM_Export.swp
   ✓ Double-click "Run_BOM_Transfer.bat"
   ✓ Check the "Processed" folder for output files

DAILY WORKFLOW:
==============

Step 1: Open assembly in SolidWorks
Step 2: Run BOM Export macro  
Step 3: Run "Run_BOM_Transfer.bat"
Step 4: Import "*_Expendable_Ready.csv" into Expendable ERP

TROUBLESHOOTING:
===============

❌ PROBLEM: "No CSV files found"
✅ SOLUTION: Run the SolidWorks macro first

❌ PROBLEM: PowerShell execution error  
✅ SOLUTION: Right-click PowerShell > Run as Administrator
           Type: Set-ExecutionPolicy RemoteSigned

❌ PROBLEM: Missing part data
✅ SOLUTION: Add custom properties to SolidWorks parts
           See: Templates\SolidWorks_Custom_Properties_Template.txt

❌ PROBLEM: Import errors in Expendable ERP
✅ SOLUTION: Check field mapping in config.json
           Follow Import_Instructions.txt
